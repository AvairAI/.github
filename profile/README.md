### What is AvairAI?

AvairAI is an **AI Sales Prospecting Platform** built for B2B. Instead of blasting a static list, we use **Pain-Signal Targeting** to find accounts showing public evidence of the problem you solve, then run multi-channel outbound with a real rep in the loop — **Pair Selling** — so you get **interested leads**, not just activity.
